### Should I self-host COMS or use the hosted service?

Feature Comparison:

| Feature                                                      | Hosted                                                                              | Self-Hosted                                                                                                                                                                                                             |
| :----------------------------------------------------------- | :---------------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Keycloak Realm                                               | SSO '[Standard Realm](https://github.com/bcgov/sso-keycloak/wiki#standard-service)' | Any OIDC realm                                                                                                                                                                                                          |
| IDP support                                                  | `IDIR`<br />`BCSC`<br />`Basic BCeID`<br />`Business BCeID`                         | Configurable                                                                                                                                                                                                            |
| [BCBox](https://bcbox.nrs.gov.bc.ca/) integration            | Yes                                                                                 | No                                                                                                                                                                                                                      |
| Hosting Platform                                             | [OpenShift](Architecture-Hosted.md#infrastructure)                                  | [Source Code](https://github.com/bcgov/common-object-management-service/)<br />[Docker](https://hub.docker.com/r/bcgovimages/common-object-management-service/)<br />[OpenShift](Architecture-Hosted.md#infrastructure) |
| Database Custodians                                          | Us                                                                                  | You                                                                                                                                                                                                                     |
| Object Storage Custodians                                    | You                                                                                 | You                                                                                                                                                                                                                     |
| [OIDC Authentication](Authentication.md#oidc-authentication) | Yes                                                                                 | Yes                                                                                                                                                                                                                     |
| [S3 Service Accounts](Authentication.md#s3-service-account)  | Yes                                                                                 | Yes                                                                                                                                                                                                                     |
| ['global' Basic Auth](Authentication.md#global-basic-auth)   | No                                                                                  | Configurable                                                                                                                                                                                                            |
| [No-Auth mode](Config.md#unauthenticated)                    | No                                                                                  | Configurable                                                                                                                                                                                                            |
| Multi-bucket support                                         | Yes                                                                                 | Yes                                                                                                                                                                                                                     |
| Strict [Privacy mode](Config.md#privacy-controls)            | Yes                                                                                 | Configurable                                                                                                                                                                                                            |
| Custom configuration options                                 | No                                                                                  | Yes                                                                                                                                                                                                                     |
